# Gym Leveling

A lightweight RPG-style workout planner that generates workouts based on your equipment, fitness level, and recovery status.

## Features

* RPG-inspired exercise progression
* Home and gym workout support
* Offline-first experience
* Workout history and progress tracking
* Installable as an app (PWA)

## Try It

https://rahmani-mahdi.github.io/gym-leveling/

## Install as an App

### Android (Chrome)

1. Open the app.
2. Tap the **Install** icon in the top address/search bar.
3. If it doesn't appear, tap **⋮ → Install app**.
4. Confirm **Install**.

### iPhone / iPad (Safari)

1. Open the app in Safari.
2. Tap **Share**.
3. Select **Add to Home Screen**.
4. Tap **Add**.

### PC (Chrome, Edge)

1. Open the app.
2. Click the **Install** icon in the address bar.

Alternatively:

* **Chrome:** Menu (**⋮**) → **Install Gym Leveling**
* **Edge:** Menu (**⋯**) → **Apps** → **Install this site as an app**

The app will be installed and can be launched like a native application.
